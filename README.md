## User credentials

Create .env.local file and add:

```
VITE_API_USERNAME=user
VITE_API_PASSWORD=password
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
